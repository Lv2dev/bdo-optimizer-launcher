검은사막 실행과 PC 최적화를 한곳에서 관리하는 작은 도구

BDO Optimizer
사용 설명서

처음 사용하는 분도 천천히 따라 할 수 있도록, 설치부터 실행, 버튼 의미, 문제가 생겼을 때 확인할 부분까지 차분하게 정리했습니다.

이 프로그램은 검은사막 게임 파일을 바꾸지 않습니다. Windows가 제공하는 권한 안에서 게임 프로세스의 우선순위와 CPU 사용 범위를 조정하고, 예약 종료와 자동 모드 전환을 도와줍니다.

버전: v0.1.0
대상: Windows 10/11 64비트
문서 기준일: 2026-06-04

이 문서는 “컴퓨터를 잘 몰라도 차근차근 따라 할 수 있는 설명서”를 목표로 작성했습니다. 모르는 용어가 나오면 그 부분은 건너뛰고, 번호가 붙은 순서만 따라 하셔도 괜찮습니다.

1. 먼저 알아두면 좋은 점

이 프로그램이 하는 일

BDO Optimizer는 검은사막을 실행하고, 게임이 켜져 있을 때 PC 자원을 조금 더 알맞게 쓰도록 돕는 프로그램입니다. 예를 들어 게임을 할 때는 고성능으로, 잠시 자리를 비우거나 창을 내려둘 때는 저전력으로 바꿀 수 있습니다.

관리자 권한이 필요한 이유

Windows에서 다른 프로그램의 우선순위나 CPU 사용 범위를 바꾸려면 관리자 권한이 필요합니다. 그래서 처음 실행할 때 “이 앱이 디바이스를 변경하도록 허용하시겠습니까?”라는 창이 나오면 예를 눌러야 합니다.

안심해도 되는 부분: BDO Optimizer는 검은사막 파일, 게임 메모리, 네트워크 패킷을 수정하지 않습니다. 다만 게임 회사의 정책은 바뀔 수 있으므로, 외부 도구 사용에 대한 책임은 사용자에게 있습니다.

2. 설치와 첫 실행

1단계. 파일 내려받기

  1. GitHub Releases 페이지에서 bdo-optimizer-launcher.exe를 내려받습니다.
  2. 같은 곳에 있는 SHA256SUMS.txt도 함께 내려받아 두면 파일이 손상되지 않았는지 확인할 수 있습니다.
  3. 가능하면 Program Files처럼 일반 사용자가 함부로 바꾸기 어려운 폴더에 exe를 두는 것을 권장합니다.

2단계. Windows 경고창 처리하기

처음 실행할 때 SmartScreen 경고가 보일 수 있습니다. 코드서명 인증서를 아직 붙이지 않았기 때문에 생기는 정상적인 경고입니다.

  1. “Windows에서 PC를 보호했습니다” 화면이 나오면 추가 정보를 누릅니다.
  2. 그다음 실행 버튼을 누릅니다.
  3. UAC 창이 나오면 예를 누릅니다.
자동 시작을 쓸 때: exe 파일이 다운로드 폴더, AppData, 임시 폴더처럼 누구나 쉽게 바꿀 수 있는 위치에 있으면 자동 시작 등록이 거부될 수 있습니다. 이 경우 exe를 더 안전한 폴더로 옮긴 뒤 다시 설정하세요.

파일이 맞는지 확인하고 싶다면

PowerShell을 열고 아래처럼 입력하면 exe의 해시 값을 볼 수 있습니다. 나온 값이 SHA256SUMS.txt에 적힌 값과 같으면 정상입니다.

Get-FileHash .\bdo-optimizer-launcher.exe -Algorithm SHA256

3. 제어 탭: 가장 자주 쓰는 화면

제어 탭은 앱의 기본 화면입니다. 게임이 켜져 있는지, 현재 어떤 모드인지, 스케줄 규칙이 있는지를 한눈에 보여줍니다.

제어 탭. 상태 확인, 게임 실행, 모드 변경을 한곳에서 할 수 있습니다.

버튼을 어떻게 쓰면 되나요?

버튼/상태 의미
상태 새로고침 게임이 켜졌는지 다시 확인합니다. 게임을 방금 켰는데 화면이 바뀌지 않으면 눌러보세요.
게임 실행 검은사막 런처를 실행합니다. 런처 경로가 잘못되어 있으면 설정 탭에서 경로를 확인합니다.
고성능 게임할 때 권장합니다. CPU 우선순위를 높이고 성능 중심으로 맞춥니다.
일반 기본에 가까운 상태입니다. 특별히 바꿀 필요가 없을 때 사용합니다.
저전력 잠시 자리를 비우거나 게임 창을 내려둘 때 사용합니다. PC 부담을 줄이는 쪽입니다.
처음에는 너무 많은 설정을 바꾸지 않아도 됩니다. 게임을 켜고, 제어 탭에서 고성능만 눌러보는 것으로 시작해도 충분합니다.

4. 스케줄 탭: 자동 모드와 예약 종료

스케줄 탭은 “언제 어떤 모드로 바꿀지”와 “언제 PC를 끌지”를 정하는 곳입니다. 매일 같은 시간에 게임을 하거나, 새벽에 자동 종료하고 싶을 때 유용합니다.

스케줄 탭. 자동 모드 전환과 PC 예약 종료를 설정합니다.

PC 예약 종료 등록하기

  1. 스케줄 탭에서 PC 예약 종료 영역을 확인합니다.
  2. 한 번만 종료하려면 단발 종료를 선택합니다.
  3. 매주 같은 요일에 종료하려면 매주 반복을 선택합니다.
  4. 날짜 또는 요일, 시간을 정한 뒤 예약 등록을 누릅니다.
  5. 취소하고 싶으면 같은 화면의 예약 취소를 누릅니다.

자동 모드 전환은 언제 쓰나요?

예를 들어 “저녁 8시부터 새벽 1시까지는 고성능, 그 외 시간은 저전력”처럼 정해둘 수 있습니다. 다만 처음 쓰는 분이라면 예약 종료만 먼저 사용해 보고, 익숙해진 뒤 자동 모드를 추가하는 것을 권장합니다.

예약 종료는 Windows 작업 스케줄러를 사용합니다. 한 번 등록하면 앱 창을 닫아도 예약은 남아 있을 수 있으니, 필요 없어졌다면 꼭 예약 취소를 눌러주세요.

5. 모니터 탭: PC 상태 보기

모니터 탭은 게임이 실행 중일 때 CPU, 메모리, GPU, VRAM, FPS 같은 정보를 보여줍니다. 숫자를 정확히 외울 필요는 없고, “너무 높게 치솟는지” 정도만 보셔도 충분합니다.

모니터 탭. 게임 실행 중일 때 자원 사용량을 그래프로 볼 수 있습니다.

무엇을 보면 되나요?

항목 쉽게 말하면
CPU 게임과 프로그램들이 CPU를 얼마나 쓰는지 보여줍니다.
메모리 RAM을 얼마나 쓰는지 보여줍니다. 여유가 너무 적으면 다른 프로그램을 닫아보세요.
GPU / VRAM 그래픽카드와 그래픽 메모리 사용량입니다. 그래픽 옵션을 높이면 올라갈 수 있습니다.
FPS 화면이 초당 몇 장씩 그려지는지 나타냅니다. 낮을수록 화면이 끊겨 보일 수 있습니다.
코어별 CPU 코어가 어떻게 쓰이는지 보여줍니다. 고성능/저전력 모드가 제대로 적용되는지 확인할 때 유용합니다.
숫자가 조금 오르내리는 것은 정상입니다. 게임 장면, 마을/사냥터, 창 전환에 따라 그래프가 계속 움직일 수 있습니다.

6. 설정 탭: 내 사용 방식에 맞추기

설정 탭은 앱의 색상, 시작 방식, 트레이 동작, 런처 경로, 업데이트 확인을 관리하는 곳입니다.

설정 탭. 테마, 트레이 동작, 자동 시작, 런처 경로 등을 관리합니다.

처음에 확인하면 좋은 설정

설정 추천
자동 (OS 설정) Windows의 밝은/어두운 테마를 따라갑니다. 특별히 취향이 없으면 그대로 두면 됩니다.
애니메이션 줄이기 화면 움직임이 불편하거나 노트북 배터리를 조금 아끼고 싶다면 켭니다.
게임이 트레이로 내려가면 자동 저전력 모드 게임 창을 내려두고 다른 일을 자주 한다면 켜두면 좋습니다.
창 닫기 시 트레이로 숨기기 실수로 앱이 꺼지는 것을 막고 싶다면 켜두세요. 완전히 종료하려면 트레이 메뉴에서 종료합니다.
Windows 시작 시 자동 실행 매일 게임을 한다면 편합니다. 불편하면 언제든 끌 수 있습니다.

트레이에서 다시 여는 방법

  1. 작업 표시줄 오른쪽 아래의 작은 화살표를 누릅니다.
  2. BDO Optimizer 아이콘을 찾습니다.
  3. 아이콘을 더블클릭하면 창이 다시 열립니다.
  4. 완전히 끄려면 아이콘을 마우스 오른쪽 버튼으로 누른 뒤 종료를 선택합니다.

7. 문제가 생겼을 때

상황 천천히 확인할 것
모드 버튼이 잘 안 눌립니다. 검은사막이 켜져 있는지 확인한 뒤 상태 새로고침을 눌러보세요. 관리자 권한으로 실행됐는지도 확인합니다.
게임 실행 버튼이 동작하지 않습니다. 설정 탭의 런처 경로가 맞는지 확인합니다. 검은사막 런처 파일 이름은 보통 BlackDesertLauncher.exe입니다.
자동 시작 등록이 안 됩니다. exe가 다운로드 폴더나 AppData 아래에 있으면 안전상 거부될 수 있습니다. 더 안전한 폴더로 옮긴 뒤 다시 시도하세요.
FPS가 보이지 않습니다. 게임이 완전히 실행된 뒤 모니터 탭을 다시 열어보세요. 그래도 안 보이면 앱을 다시 실행해 봅니다.
창을 닫았는데 앱이 꺼지지 않았습니다. 설정에서 “창 닫기 시 트레이로 숨기기”가 켜져 있으면 정상입니다. 작업 표시줄 오른쪽 아래 트레이에서 다시 열거나 종료할 수 있습니다.
문제가 계속됩니다. 설정 탭의 로그 폴더 열기를 눌러 로그 파일을 확인합니다. 오류를 공유할 때 이 로그가 도움이 됩니다.

8. 마무리

처음에는 많은 기능을 한 번에 쓰려고 하지 않아도 됩니다. 먼저 게임을 실행하고, 제어 탭에서 고성능/일반/저전력 버튼이 어떤 느낌인지 확인해 보세요. 익숙해진 뒤 예약 종료, 자동 모드 전환, 자동 시작 같은 기능을 하나씩 켜면 됩니다.
